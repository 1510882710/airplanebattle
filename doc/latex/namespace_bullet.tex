\doxysubsection{详细描述}
\begin{DoxyVerb}单个子弹类
    1. move 子弹移动函数，根据具体的子弹发射策略来指定子弹的移动方向
    2. draw_element 绘制子弹对象
    3. set_unused 当子弹无效时设置可不见，并将其设置到一个不合法的位置上
    4. set_used 当子弹发射时设置可见，并设置初始位置
\end{DoxyVerb}
